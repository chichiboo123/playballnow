# 어디서나 야구장 (Playball Now) ⚾

실제 야구장 전광판을 그대로 재현한 **단일 HTML 파일** 디지털 스코어보드 웹 애플리케이션입니다.

## 📋 프로젝트 개요

`index.html` 하나로 동작하는 독립 실행형 야구 스코어보드입니다. 별도의 빌드 과정이나 서버 없이 브라우저에서 파일을 열기만 하면 경기 기록·선수 관리·타석 결과 추적·데이터 내보내기 등 모든 기능을 사용할 수 있습니다. 효과음(오디오)까지 파일 내부에 내장되어 있어 핵심 스코어보드 기능은 오프라인에서도 작동합니다. (단, PDF/JPG 내보내기는 CDN으로 불러오는 `jsPDF`·`html2canvas` 라이브러리를 사용하므로 인터넷 연결이 필요합니다.)

## ✨ 주요 기능

### 🎮 경기 관리
- **전광판(스코어보드)**: 1~11회 이닝별 득점과 R(득점)/H(안타)/E(실책)/B(볼넷) 합계 표시
- **이닝 이동**: ◀ / ▶ 버튼으로 초·말 전환 및 이닝 이동, 현재 이닝 하이라이트
- **BSO 카운터**: 볼(B)·스트라이크(S)·아웃(O) 인디케이터 클릭으로 실시간 카운트
  - **타자교체**: 볼/스트라이크만 초기화
  - **공수교대**: 볼/스트라이크/아웃 모두 초기화
- **주자 표시판**: 다이아몬드 위 1·2·3루 및 홈 베이스에 주자 위치 시각화

### 🧾 타석 결과 기록
선수별 타석 결과를 12가지 옵션으로 기록합니다.
- 아웃, 1루타, 2루타, 3루타, 홈런
- 뜬공, 땅볼, 병살타
- 볼넷, 사구, 상대실책
- 삭제(기록 취소)

홈런 기록 시 득점이 자동으로 연동되며, 결과에 따라 안타(H)·실책(E)·볼넷(B) 통계가 자동 집계됩니다.

### 👥 선수 관리
- HOME / AWAY 팀별 라인업 구성
- **선수 추가 / 삭제** 및 선수 번호·이름 편집
- 팀 이름 직접 입력(편집)
- 선수별 타석 결과 누적 기록

### 💾 데이터 관리
- **자동 저장**: 브라우저 `localStorage`에 게임 상태 자동 저장 → 새로고침해도 기록 유지
- **백업 다운로드 / 업로드**: JSON 파일로 게임 데이터 저장·복원 (헤더 우측 아이콘)
- **내보내기**: TXT(경기 요약), PDF(문서), JPG(전광판 캡처)
- **초기화**: 확인 모달을 거쳐 게임 상태 완전 리셋

### 🎵 사운드 효과
- 3가지 야구장 응원가 내장 (파일에 base64로 포함)
- 재생/일시정지 토글 (한 번에 하나만 재생)
- 음량 슬라이더 조절
- 음원 출처: [www.101soundboards.com](https://www.101soundboards.com/)

## 🚀 사용 방법

별도 설치가 필요 없습니다. `index.html` 파일을 브라우저로 열기만 하면 됩니다.

```bash
# macOS
open index.html

# Windows
start index.html

# Linux
xdg-open index.html
```

GitHub Pages 등 정적 호스팅에 그대로 올려서 사용할 수도 있습니다.

## 🎨 디자인 특징

- **전광판 스타일**: 실제 야구장 스코어보드를 닮은 어두운 테마
- **색상 테마**
  - 페이지 배경: 연한 회색 (`#f3f4f6`)
  - 스코어보드/패널: 진한 검정·회색 (`#1a1a1a`, `#1f2937`)
  - 강조: 네온 그린·오렌지·레드, 필드 그린
- **반응형 레이아웃**: 모바일·태블릿·데스크톱 대응
  - 좁은 화면(모바일)에서는 전광판 → 컨트롤 → 라인업 순으로 세로 배치
  - 768px 이상에서는 컨트롤 패널과 전광판을 좌우 2열로 배치
- **한글 폰트**: Do Hyeon, Sunflower, Nanum Gothic, Orbitron (Google Fonts)
- **접근성**: 주요 버튼에 `aria-label` / `aria-pressed` 제공

## 🔧 기술 스택

- 순수 **HTML5 / CSS3 / Vanilla JavaScript** (프레임워크·빌드 도구 없음)
- 상태 저장: 브라우저 `localStorage`
- 화면 캡처/문서화: `html2canvas` 1.4.1, `jsPDF` 2.5.1 (CDN, JPG·PDF 내보내기에 사용)
- 폰트: Google Fonts (Do Hyeon, Sunflower, Nanum Gothic, Orbitron)

## 📦 내보내기 형식

| 형식 | 설명 |
| --- | --- |
| **TXT** | 팀·점수·타석 결과가 정리된 경기 요약 텍스트 |
| **PDF** | 전광판 화면을 담은 문서 형식 |
| **JPG** | 전광판 스크린샷 이미지 |
| **JSON** | 전체 게임 데이터 백업(복원 가능) |
