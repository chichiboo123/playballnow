# 어디서나 야구장 (Playball Now)

디지털 야구 스코어보드 웹 애플리케이션

## 📋 프로젝트 개요

실제 야구장 전광판을 그대로 재현한 디지털 스코어보드 애플리케이션입니다. 경기 기록, 선수 관리, 타석 결과 추적, 데이터 내보내기 등 야구 경기에 필요한 모든 기능을 제공합니다.

## ✨ 주요 기능

### 🎮 경기 관리
- **실시간 스코어보드**: 이닝별 득점 현황 표시  
- **BSO 카운터**: 볼/스트라이크/아웃 카운트 실시간 업데이트  
- **주자 표시**: 베이스에 주자 위치 시각화  
- **타석 결과 기록**: 12가지 타석 결과 옵션  
  - 아웃, 1루타, 2루타, 3루타, 홈런  
  - 뜬공, 땅볼, 병살타  
  - 볼넷, 사구, 상대실책, 삭제  

### 👥 선수 관리
- 팀별 선수 라인업 구성  
- 선수 번호 및 이름 설정  
- 타순 관리 및 현재 타자 표시  
- 선수별 타석 결과 기록  

### 📊 통계 기록
- 이닝별 득점 집계  
- 안타(H), 에러(E), 볼넷(BB) 자동 계산  
- 타석 결과별 통계 자동 업데이트  

### 💾 데이터 관리
- **내보내기**: TXT, PDF, JPG 형식 지원  
- **백업/복원**: JSON 형식으로 게임 데이터 저장/불러오기  
- **초기화**: 게임 상태 완전 리셋  

### 🎵 사운드 효과
- 3가지 야구장 응원가 내장  
- 재생/일시정지 토글 기능  
- 한 번에 하나의 음악만 재생  

## 🚀 사용 방법

### 독립 실행형 HTML 버전
`baseball_scoreboard.html` 파일을 브라우저로 열기만 하면 됩니다.  
**인터넷 연결이나 별도 설치 없이 모든 기능이 작동**합니다.

```bash
# 파일 열기
open baseball_scoreboard.html
```

### React/TypeScript 개발 버전

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 프로덕션 빌드
npm run build
npm start
```
